# Using PlatformIO with ESP32
This is a quick installation guide to help you set up PlatformIO to work with the ESP32

## Creating a Project
Once you have fully installed PlatformIO into your VS code, (see [here](https://docs.platformio.org/en/latest/integration/ide/vscode.html#installation)). Then open the window on the left, and create a new project. For the board, select `Espressif ESP32 Dev Module`.
